# 人人看得懂的 Claude Code 拆解

> 这里的文章不需要你懂代码。
>
> 你只需要用过 ChatGPT 或 Claude，对 AI 工具有一点点好奇心。

---

## 这是什么

这是 [`docs/book/chapters/`](../chapters/) 技术版的配套读物。

技术版面向会看源码的人，讲的是"代码里怎么实现的"。

这里面向会用 AI 工具但不看代码的人，讲的是"为什么要这么设计，这对你意味着什么"。

两个版本的结论是一致的，只是讲法不同。
