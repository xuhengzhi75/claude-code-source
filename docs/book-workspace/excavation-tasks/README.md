# 源码挖掘任务卡

本目录是 startheart（源码分析侧）和小许（写作侧）之间的交接区。

## 协作流程

1. 小许识别哪些章节内容薄弱，起草挖掘任务卡（`ch**-excavation.md`）
2. startheart 认领任务卡，深挖源码后把结论填入对应卡片，commit 提交
3. 小许读取填好的卡片，据此扩充/重写章节正文，commit 提交
4. xuhengzhi75 验收

## 填写规范

每条结论必须标注来源类型：
- `verified`：有明确源码行号支撑
- `inference`：基于源码的合理推断，但无直接证据

填写时请附上：文件路径 + 行号范围 + 关键代码片段（可截取关键3-5行）
