# Planning

存放书项目的规划、排期、进度与写作队列类文档。

## 先按任务选入口

### 1) 看当前总状态
1. `project-status.md`
2. `completed-coverage-and-writing-queue.md`

### 2) 看阶段目标与边界
1. `roadmap.md`
2. `requirements.md`

### 3) 看协作与推进方式
1. `collab-split.md`
2. `progress-management.md`

## 文件职责
- `project-status.md`：当前总状态、风险、下一步
- `completed-coverage-and-writing-queue.md`：已完成 / 待写清单
- `roadmap.md`：阶段目标与里程碑
- `requirements.md`：验收标准与范围约束
- `collab-split.md`：协作分工
- `progress-management.md`：推进与跟踪方式
- `book-outline-v1.md`：提纲草稿

## 使用原则
- 想继续推进，先看 `project-status.md`
- 想判断做什么优先，先看队列和 roadmap
- 想避免偏题，先看 requirements
