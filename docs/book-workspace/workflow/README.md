# Workflow

存放执行流程、协作方法、模板与提示词类文档。

## 先按任务选入口

### 1) 需要交接给下一个写作者 / agent
1. `analysis-handoff-template.md`
2. `collaboration-guide.md`

### 2) 需要提示词或工作模板
1. `prompt-templates.md`
2. `workflow-templates.md`
3. `agent-role-templates.md`

### 3) 需要理解当前工作法
1. `WORKING-METHOD.md`
2. `collaboration-guide.md`

## 文件职责
- `WORKING-METHOD.md`：当前项目工作法
- `analysis-handoff-template.md`：分析交接模板
- `prompt-templates.md`：提示词模板
- `workflow-templates.md`：工作流模板
- `agent-role-templates.md`：agent 角色模板
- `collaboration-guide.md`：协作规范

## 使用原则
- 要交接时优先用模板，不临时自由发挥
- 需要复用动作时优先从模板出发，不重复发明流程
