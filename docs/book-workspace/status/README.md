# Status Snapshots

这个目录存放阶段性状态快照（`status-*.md`）和状态索引。

## 先按任务选入口

### 1) 想快速恢复现场
1. `status-index.md`
2. 最近一条 `status-*.md`
3. 再回看 `../planning/project-status.md`

### 2) 想追某一时段的推进变化
按时间顺序查看 `status-*.md`

## 文件职责
- `status-index.md`：快照索引与汇总入口
- `status-*.md`：某一时间点的原始状态记录

## 约定
- 新状态文件统一写入本目录
- 文件名保持 `status-YYYY-MM-DD-HHMM.md`
- `status-index.md` 负责汇总，不替代原始快照
